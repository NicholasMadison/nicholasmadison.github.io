Project Overview

Strides of Strengths

  • This is an informative website to share information on what Strides of Strength (SOS) is and how it helps members of the community.
  • The intended users are potential clients, volunteers, and other community members interested in the non-profit.
  • This website will allow users to sign-up as volunteers, donate, and learn about the programs offered at SOS.

Dynamic Functionality on the website

I will utilize event listeners and DOM manipulation effectively. For instance, on the Programs Page, I could use mouseover and mouseout events to handle the dropdown sections. When a user hovers over a program section, JavaScript could dynamically update the CSS display property of the content to make it visible. Then, when the mouse leaves, you can revert the display property to hide the content again. Similarly, for the Donation Page, I can use a click event listener on the “Donate” button to toggle the visibility of the donation form by adding or removing a class, such as “visible”, which controls the display of the form.

For interactive elements like the Volunteer Page's form submission, I can use the submit event to capture the form data and send it to my server or process it locally. JavaScript libraries like Fetch API can help you handle HTTP requests for submitting the form data securely. Additionally, I can validate the form fields in real-time using event listeners on input or change events, ensuring that users enter the required information correctly before submitting. By combining these techniques, I can create a seamless and interactive experience tailored to my site's requirements.
